-------------------------------------------------------
    Features
-------------------------------------------------------

Morning includes the following features:

    * Custom logo
    * Social media icons
    * Comment display controls
    * Custom excerpt length
    * Show full posts on Blog
    * Custom CSS

Morning also has a premium plugin upgrade called Apex Pro that includes the following features:

    * Custom colors
    * Featured Videos
    * Featured Sliders
    * New Layouts
    * Featured Image Sizes
    * Flexible header image
    * Google fonts
    * More widget areas
    * Display controls
    * Custom footer text

Apex Pro: https://www.competethemes.com/apex-pro/

-------------------------------------------------------
    Additional Info:
-------------------------------------------------------

    * All menus in Morning can have 1-3 tiers
    * Featured Images on Posts and Pages have a 2x1 ratio at all screen sizes
    * Morning is child theme of the Apex theme
